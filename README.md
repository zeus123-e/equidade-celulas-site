# Equidade no Ensino das Celulas Vivas

Site do projeto de biologia **Equidade no Ensino das Celulas Vivas**, desenvolvido para apresentar um tabuleiro sensorial inclusivo sobre celulas, corpo humano, acessibilidade e sustentabilidade.

O projeto transforma o estudo das celulas em uma experiencia visual e tatil, usando um jogo educativo com braille, texturas, cores, materiais reaproveitados e situacoes relacionadas a saude celular.

## Sobre o projeto

O tabuleiro sensorial foi criado para tornar o aprendizado mais acessivel, participativo e divertido. Durante o percurso, os estudantes encontram casas sobre habitos saudaveis, como alimentacao equilibrada, hidratacao e exercicios fisicos, alem de fatores que podem prejudicar as celulas, como infeccoes, inflamacoes, contaminacoes, mutacoes geneticas e deficiencia energetica.

## Recursos do site

- Visual moderno com dois temas: claro e escuro.
- Animacao de particulas no fundo.
- Animacoes fortes de scroll, com elementos entrando ao subir e descer.
- Galeria com fotos reais do processo e do resultado final.
- Secoes sobre proposta, acessibilidade, sustentabilidade e creditos.
- Layout responsivo para celular e computador.

## Acessibilidade

O projeto valoriza o uso de braille, texturas e contraste visual para que pessoas com deficiencia visual tambem possam participar da atividade e compreender o conteudo.
